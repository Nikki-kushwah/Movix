# Movix 🎬

Movix is a modern, responsive web application designed for movie and TV show enthusiasts. It allows users to explore trending titles, discover detailed information about films and television series, watch trailers, and search for their favorite content. 

Built using **React**, **Vite**, and **SCSS**, this application delivers a fast and fluid user experience by leveraging a robust external movie database API.

---

## 🚀 Features

* **Trending & Popular Sections:** Discover what’s hot right now in movies and TV shows.
* **Advanced Search:** Search for any movie or TV series with real-time results.
* **Detailed Insights:** View ratings, genres, release dates, runtimes, cast members, and official trailers.
* **Dynamic Carousels:** Smoothly browse through recommendations and similar content.
* **Fully Responsive:** Optimized for a seamless experience across mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

* **Frontend Framework:** React (built with [Vite](https://vitejs.dev/))
* **Styling:** SCSS / SASS (Modular and responsive design)
* **State Management / Fetching:** Axios / Hooks
* **Icons & Components:** React Icons, Circular Progress Bars
